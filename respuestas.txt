¿Qué es RVM (Ruby Version Manager)? Ruby Version Manager, comúnmente abreviado como RVM, 
es una plataforma software para sistemas basados en UNIX diseñado para administrar múltiples
instalaciones de Ruby en la misma máquina.

¿Cuáles son las diferencias entre la ejecución del programa "Hola Mundo" con el intérprete de Ruby (ruby helloWorld.rb) y el intérprete interactivo del ejercicio anterior? 
Con el intérprete de Ruby se ha de especificar el fichero que contiene el código fuente en Ruby 
que se desea ejecutar, mientras que con el intérprete interactivo se pueden introducir las 
sentencias que se quieren ejecutar desde el 'prompt'.

¿Cuáles son las diferencias entre la ejecución del programa "Hola Mundo" con el intérprete 
interactivo y con PRY?
Con PRY, se muestran más detalles a cerca de la ejecución del programa.
